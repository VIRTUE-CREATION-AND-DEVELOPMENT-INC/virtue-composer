# @virtuecreation/eslint-config-composer

ESLint boundaries for projects initialized with Virtue Composer.

The default flat configuration blocks direct package imports and raw controls, keeps `Section` structural with explicit semantics, recommends specialized field components, prefers minor-unit `Money` values, and warns when `SegmentedControl` is used as a submitted `Form` choice.

```js
import { composer } from "@virtuecreation/eslint-config-composer";

export default [composer];
```
